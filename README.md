Project Title

Marketing Funnel & Conversion Performance Analysis

Objective

Analyze user behavior across the marketing funnel (View → Cart → Purchase) to identify drop-off points and improve conversion rates.

Tools Used

Power BI
Data Visualization
Marketing Funnel Analysis

Key Insights

Major drop-off occurs between product view and cart stage

High-performing brands generate most purchases

Cart users have higher likelihood of completing purchases

Recommendations

Improve product page experience

Use retargeting strategies for visitors

Focus marketing campaigns on high-performing brands
